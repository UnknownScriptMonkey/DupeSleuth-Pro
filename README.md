🧩 DupeSleuth Pro  
Smart Duplicate & Similar Media Finder for Windows

DupeSleuth Pro helps you clean up massive folders full of **photos and videos** by automatically finding **duplicate** and **visually similar** files — even if they’ve been renamed, resized, or re-encoded.

---

## 🚀 Features

- 🔍 **Exact duplicates** via SHA-256 hashing (optional byte-verify)
- 🧠 **Similar media detection** with perceptual hashing (finds resized or re-encoded files)
- 🖼️ Scans `.jpg`, `.jpeg`, `.png`, `.gif`, `.mp4`, `.mov`
- 🧩 **Smart auto-selection rules:**  
  Keep **newest**, **oldest**, **largest**, or **shortest path**
- 📸 Thumbnail view with **image**, **GIF**, and **video** previews  
  (videos use `ffmpeg.exe` for still frames)
- 🗂️ Move duplicates to another folder or delete safely to the Recycle Bin
- ⚡ Multithreaded, NVMe-optimized performance
- 🪟 100% offline — portable, no install or internet required

---

## 🧰 Prerequisites

**Required**
- Windows 10 or 11 (64-bit)
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)  
  *(If you build as self-contained, users don’t need this.)*
